# Oprema za Mobitele - Web Shop Project

A multi-page e-commerce website for mobile accessories built with HTML5, CSS3, and JavaScript.

## Features
- **Responsive Design:** Viewable on mobile and desktop.
- **Shopping Cart Logic:** Basic "Add to Cart" functionality with a counter.
- **Interactive Slideshow:** Featured products on the homepage.
- **Contact & Login Forms:** Clean UI for user interaction.

## Built With
- HTML5 (Semantic tags)
- CSS3 (Flexbox & Custom Styling)
- JavaScript (DOM Manipulation)
